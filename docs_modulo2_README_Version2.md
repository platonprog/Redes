Convenciones de unidades SI
- Throughput/Goodput: bit/s (bps)
- Latencia/Jitter/RTT: s (se puede expresar en ms/µs para lectura)
- Ancho de banda espectral: Hz
- Velocidad de señalización: símbolos/s (baud)
- Distancia: m
- Velocidad de propagación: m/s
- Potencia: W; Energía por bit E_b: J; Densidad de ruido N_0: W/Hz (= J)
- Probabilidades, eficiencias, ventanas: adimensional
