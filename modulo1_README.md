Convenciones de unidades SI
- Velocidades: bit/s (bps), símbolos/s (baud)
- Frecuencia y ancho de banda espectral: Hz
- Distancia: m; velocidad de propagación: m/s
- Tiempo/retardos/latencia/jitter/RTT: s (ms/µs para lectura)
- Potencia: W; energía por bit E_b: J; densidad de ruido N_0: W/Hz (= J)
- Ganancias/pérdidas: dB (adimensional)
- Probabilidades/eficiencias/ventanas: adimensional
