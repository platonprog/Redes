# Capítulo 2. Teoría de Señal

Señales
- Analógicas (continuas) vs digitales (discretas en tiempo/niveles).
- Periódicas vs aperiódicas.

Sinusoidal
- $s(t)=A\cos(2\pi f t + \varphi)$: amplitud A (dependiente: V, A, Pa…), frecuencia f (Hz), fase φ (rad), longitud de onda $\lambda=v/f$ (m), velocidad de propagación v (m/s).

Potencia y decibelios
- Potencia media P (W). Relaciones en dB: $G_{dB}=10\log_{10}(P_2/P_1)$.
- Atenuación dB positiva; ganancia dB positiva. Convertir a lineal: $G=10^{G_{dB}/10}$.

Espectro
- Densidad espectral de potencia S(f) [W/Hz].
- Paso-bajo vs paso-banda; señales periódicas → espectro discreto (armónicos).

Relación señal–ruido
- $SNR=P_s/P_n$ (adim, dB si log). Ruido blanco $N_0$ (W/Hz); $P_n=N_0 B$ (W).
